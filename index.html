Gestão de Dados

Este site foi criado com a intenção de ensinar estudantes do ensino médio a trabalhar com gráficos interativos, utilizando dados reais sobre os doces mais votados no Brasil. Ele emprega tecnologias como HTML, CSS, JavaScript e a biblioteca Plotly para transformar o aprendizado de visualização de dados em uma experiência acessível e envolvente.

A estrutura do site é simples, começando com um cabeçalho que apresenta o título "Os doces brasileiros" e uma barra de navegação com links que levam a outras páginas, como "Mundo" e "Minha escola". O foco principal está na seção de gráficos, onde os estudantes podem visualizar informações de forma clara e interativa. Essa seção é onde os gráficos são gerados, mostrando quais doces foram mais votados no país.

A criação dos gráficos é feita através de um script JavaScript que consome dados de uma API externa, especificamente de um arquivo JSON hospedado no GitHub. Esses dados são então transformados em gráficos de barras utilizando a biblioteca Plotly, uma ferramenta que permite a criação de gráficos interativos de maneira simples e visualmente atraente. A interação com os gráficos torna o aprendizado mais dinâmico, permitindo que os alunos explorem diferentes dados de forma prática.

O site utiliza uma abordagem modular no código JavaScript, com funções importadas de arquivos separados. Por exemplo, o arquivo comum.js centraliza funções que ajudam na configuração de cores e fontes, garantindo consistência visual nos gráficos. O script votosPorDoces.js é responsável por gerar o gráfico principal que mostra a votação dos doces no Brasil, enquanto outros scripts, como docesNorte.js e docesSul.js, criam gráficos específicos para regiões do país.

Além disso, o design visual é estilizado com um arquivo CSS externo, e as cores e fontes utilizadas no gráfico são configuradas dinamicamente pelo JavaScript, criando um visual personalizado e atraente. Por exemplo, as barras dos gráficos são coloridas em um tom de violeta, enquanto o plano de fundo dos gráficos adota cores como vermelho e rosa, conferindo um visual vibrante e adequado para o público jovem.

A proposta educacional do site é clara: fornecer uma plataforma onde os alunos possam aprender conceitos básicos de visualização de dados de forma prática, ao mesmo tempo em que se familiarizam com técnicas de programação e o uso de bibliotecas como Plotly. Eles podem experimentar a criação e manipulação de gráficos, compreendendo como alterações no código afetam diretamente a apresentação visual dos dados. Com isso, o site incentiva o desenvolvimento de habilidades que são úteis em diversas áreas do conhecimento, como matemática, estatística e ciência da computação.

A proposta é criar uma experiência onde o usuário toma decisões em uma história fictícia que envolve a preservação da Mata Atlântica, um bioma brasileiro. Ao longo do site, o usuário é apresentado a uma série de perguntas e, a cada escolha, a história evolui de maneira diferente, resultando em um desfecho personalizado.

A estrutura HTML define a base da página, com uma caixa principal onde as perguntas, alternativas de resposta e o resultado final são exibidos. O estilo visual do site é controlado por um arquivo CSS externo, responsável pela aparência dos elementos, como o espaçamento, cores e layout, oferecendo uma interface amigável ao usuário.

O JavaScript é o motor que possibilita a interatividade. Ele gerencia o conteúdo dinâmico da página, exibindo as perguntas, registrando as escolhas do usuário e montando a narrativa final com base nas respostas. A cada pergunta, alternativas são apresentadas como botões interativos. Quando o usuário seleciona uma opção, uma nova etapa da história se desenrola, e no final do quiz, o JavaScript exibe um resultado personalizado, refletindo as consequências das escolhas feitas ao longo do processo.

Ao mesmo tempo, a narrativa em torno da Mata Atlântica serve para reforçar conceitos de tomada de decisão e interatividade, ao mesmo tempo que explora questões ambientais importantes.

Jogo da vaquinha

Esse site é um exercício interativo que tem como objetivo ensinar lógica de programação utilizando JavaScript e a biblioteca p5.js, por meio da criação de um jogo 2D simples. No jogo, o jogador controla um personagem, chamado "ator", que deve atravessar uma estrada movimentada por carros. O objetivo principal é fazer com que o ator cruze a estrada com sucesso, evitando colisões com os veículos. Cada vez que o jogador consegue atravessar a pista, ele ganha um ponto, mas, se colidir com algum carro, o personagem é enviado de volta ao ponto de partida, e o jogador perde um ponto, desde que tenha mais de zero.

A movimentação do ator é feita por meio das setas do teclado, permitindo que o personagem se mova para cima e para baixo. Caso o jogador tente mover o ator para fora da tela na parte inferior, o movimento é limitado, garantindo que o ator não saia dos limites do jogo. Os carros se deslocam horizontalmente pela tela, simulando um tráfego constante, e quando saem do campo de visão, eles são reposicionados no início da estrada, criando um loop contínuo de veículos passando pela tela.

A detecção de colisões é feita por uma função específica, que utiliza a biblioteca para identificar se o ator (representado como um círculo) colidiu com algum dos carros (representados como retângulos). Quando isso acontece, o ator retorna ao ponto inicial, e um som de colisão é tocado, indicando o evento, além de causar a perda de pontos do jogador, caso este tenha uma pontuação positiva.

Além da mecânica de pontuação, o jogo conta com sons para dar um retorno auditivo ao jogador, como a música de fundo, o som de colisão e um efeito sonoro quando o jogador ganha pontos. Os gráficos do jogo são simples, consistindo em uma imagem de uma estrada para o cenário e sprites representando tanto o ator quanto os carros que cruzam a tela.

Esse exercício serve como uma ferramenta de aprendizado para introduzir conceitos de lógica de programação, como controle de fluxo, manipulação de arrays, criação de funções e detecção de colisões, ao mesmo tempo em que oferece uma introdução à programação gráfica com p5.js.

Labirinto

O projeto de criação de um labirinto interativo no Scratch foi desenvolvido como parte de uma introdução à lógica de programação, com o intuito de incentivar o aprendizado prático e intuitivo dos conceitos fundamentais de programação e desenvolvimento de jogos. Utilizando a plataforma Scratch, uma ferramenta voltada para o ensino de programação de forma visual e amigável, os estudantes puderam explorar a criação de um jogo interativo de labirinto.

Os principais objetivos do projeto incluíram a introdução de conceitos básicos de programação, permitindo que os alunos tivessem contato inicial com sequências de comandos, loops, condicionais e eventos, todos aplicados na criação do jogo. Além disso, o projeto visou desenvolver o raciocínio lógico, uma vez que resolver problemas, como conduzir um personagem através de um labirinto, exigia pensar de forma lógica e dividir as tarefas em pequenas etapas.

O jogo foi além da simples movimentação dos personagens, pois os alunos programaram interações entre eles e o cenário. Quando um personagem colidia com uma parede, o aluno precisava programar a lógica para que ele voltasse ou tentasse outra rota. As regras de vitória e derrota também foram implementadas, estabelecendo que o jogo terminaria quando o personagem chegasse ao final do labirinto ou falhasse.

Além de promover aprendizado colaborativo, onde os alunos trabalharam tanto individualmente quanto em grupos, compartilhando ideias e soluções, o projeto também ajudou a aplicar conceitos matemáticos. Ao programar o movimento do personagem, os estudantes utilizaram noções de coordenadas cartesianas, ângulos de rotação e distâncias, reforçando assim seu aprendizado em matemática. O ambiente do Scratch proporcionou feedback imediato, permitindo que os alunos testassem seu código em tempo real, identificando e corrigindo erros de programação, o que estimulou um ciclo de tentativa e erro que favoreceu a perseverança e o pensamento crítico.

Em conclusão, o projeto de criação do labirinto no Scratch provou ser uma ferramenta eficaz para introduzir os conceitos iniciais de programação de maneira acessível e divertida. Além de desenvolver habilidades técnicas, como a criação de algoritmos simples e o uso de comandos, os alunos também aprimoraram sua capacidade de resolução de problemas e trabalharam a criatividade por meio da personalização do jogo. Este tipo de atividade não só contribui para o desenvolvimento acadêmico, mas também prepara os alunos para desafios futuros em áreas relacionadas à tecnologia e inovação.
